Projeto de Classificao Supervisionada com Scikit-Learn: Identificao de Vinhos Doces e Secos

Este projeto foi criado para aprender a utilizar modelos supervisionados de classificao do Scikit-Learn, especificamente o RandomForestClassifier, aplicados a um dataset de vinhos. O objetivo foi treinar um modelo capaz de identificar vinhos doces e secos com base em algumas de suas caractersticas qumicas.

Durante a implementao, foi possvel observar que, utilizando os dados disponveis, a maioria dos vinhos doces apresenta uma intensidade de cor maior em comparao aos vinhos secos. Essa descoberta demonstra uma possvel relao entre a doura de um vinho e sua colorao mais intensa.

O projeto serviu como uma excelente introduo ao uso de algoritmos de aprendizado supervisionado e reforou a importncia da anlise exploratria de dados para identificar padres relevantes.

wines = load_wine()

df = pd.DataFrame(wines["data"], columns=wines["feature_names"])
df['target'] = wines.target
df

# Criar um rtulo binrio (doce vs seco) como exemplo
# Digamos que 'classe 2' seja mais doce (isso  fictcio)
df['sweetness'] = df['target'].apply(lambda x: 1 if x == 2 else 0)

# Dividir em features (X) e label (y)
X = df.drop(['target', 'sweetness'], axis=1)
y = df['sweetness']


# Dividir em treino e teste
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=178, stratify=y)

# Escalar os dados
scaler = StandardScaler()
X_train_scaled = scaler.fit_transform(X_train)
X_test_scaled = scaler.transform(X_test)

# Treinar o modelo
rf_model = RandomForestClassifier(random_state=178)
rf_model.fit(X_train_scaled, y_train)

# Predies
y_pred = rf_model.predict(X_test_scaled)

# Relatrio de classificao
print(classification_report(y_test, y_pred))

# Matriz de confuso
cm = confusion_matrix(y_test, y_pred)
sns.heatmap(cm, annot=True, fmt='d', cmap='Blues')
plt.xlabel('Predicted')
plt.ylabel('Actual')
plt.title('Confusion Matrix')
plt.show()

feature_importances = pd.DataFrame(rf_model.feature_importances_,

print(feature_importances)

# Visualizar as importncias
feature_importances.plot(kind='bar', figsize=(10, 6))

# Separar os vinhos doces e secos
sweet_wines = predicted_df[predicted_df['Predicted'] == 1]
dry_wines = predicted_df[predicted_df['Predicted'] == 0]

# Escolher features para visualizao
features_to_plot = ['alcohol', 'color_intensity']

# Calcular as mdias para cada grupo
sweet_means = sweet_wines[features_to_plot].mean()
dry_means = dry_wines[features_to_plot].mean()

# Criar grfico de barras
x = range(len(features_to_plot))
plt.bar(x, sweet_means, width=0.4, label='Doces', align='center')
plt.bar([i + 0.4 for i in x], dry_means, width=0.4, label='Secos', align='center')

# Personalizar o grfico
plt.xticks([i + 0.2 for i in x], features_to_plot, rotation=45)
plt.ylabel('Mdia das Features')
plt.title('Comparao de Features: Vinhos Doces vs Secos')
plt.legend()
plt.tight_layout()
